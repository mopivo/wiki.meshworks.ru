## Заголовки и текст

Обычный текст с **жирным** и _курсивом_. Можно добавлять списки:

- Короткие пункты.
- Пункты с описанием.
  - Вложенный список.

И таблицы:

| Модель | Диапазон | Время работы |
|--------|----------|--------------|
| Heltec T114 | 868 МГц | ~24 часа |
| MeshPocket | 915 МГц | ~36 часов |

```markdown
:::tip
Показываем подсказки и лайфхаки.
:::

:::info
Оповещаем о важной детали.
:::

:::warning
Предупреждаем об ошибке или ограничении.
:::

:::favorite
```

:::

## Код и команды

Короткие команды выделяем одинарными обратными кавычками: `` `meshtastic --help` ``.

Длинные блоки — тройными с указанием языка:

```bash
meshtastic --set owner "MeshWorks"
meshtastic --ch-add longfast --psk mysecretkey
```

## Встраивание компонентов

Можно подключать React-компоненты и переиспользуемые блоки. Простой пример — собственный компонент кнопки:

```mdx
import Link from '@docusaurus/Link';

<Link className="button button--primary" to="/devices">
  Перейти к устройствам
</Link>
```

> Не бойтесь экспериментировать. Если что-то не отрендерилось — `npm start` или превью в PR покажет ошибку.
